ICLVS — Intelligent Contextual Legal Verification System

Pipeline reproducible para verificación legal contextual: Ingestión → Razonamiento → Adaptación normativa → Trazabilidad. Minimal, técnico y listo para GitHub Pages.

Proveniencia Gráficas

Resumen técnico (versión 0.1.0)

El prototipo ICLVS modela un flujo verificable donde cláusulas y requisitos se mapean a marcos legales aplicables y producen salidas auditables (reglas, motivos, referencias). Incluye diagramas vectoriales, matriz de trazabilidad y métricas demostrativas; el motor automatizado no está incorporado en esta versión.

DOI / SSRN: 10.2139/ssrn.5646790 · Perfil autor: SSRN (Espinal De Aza)
